Tables 

Tables are used to display in terms rows and columns. 
Table tag used structure in html . 
Rows for the table are created using <tr> tag. 
<th> tag used to display the title display 
The values for the title using <td> tag.

        where rows span is used to cocupy more then one row colspan 
        occupy more then 1 column
